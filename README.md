# Parole Crociate Governance DOS

Cruciverba interattivo statico, open source e senza dipendenze esterne.

## Pubblicazione con GitHub Pages

1. Crea un nuovo repository pubblico su GitHub, ad esempio `governance-dos-crossword`.
2. Carica nella radice del repository i file `index.html`, `style.css`, `app.js` e `puzzle.json`.
3. Apri **Settings → Pages**.
4. In **Build and deployment**, scegli **Deploy from a branch**.
5. Seleziona il branch `main` e la cartella `/ (root)`, quindi salva.
6. GitHub mostrerà il link pubblico del sito, normalmente:
   `https://TUO-USERNAME.github.io/governance-dos-crossword/`

## Modifica delle domande

Apri `puzzle.json`. Ogni voce contiene risposta, definizione, coordinate, direzione e numero.

## Dataset per Crossword Labs

Il file `dataset-crossword-labs.txt` contiene una riga per parola nel formato:

`RISPOSTA<TAB>definizione`

Può essere incollato nell'editor di Crossword Labs.
